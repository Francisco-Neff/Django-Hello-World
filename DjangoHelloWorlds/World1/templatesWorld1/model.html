Definición de modelos

A través de "models.py" creado con la aplicación, donde podemos ver un ejemplo aquí, podemos crear una tabla con los atributos que definamos en la clase, como si se tratara de una programación orientada a objetos. 

    - Model en models.py  
        
        class Modelo1(models.Model):
            campo1 = models.CharField(max_length=30)
            campo2 = models.CharField(max_length=30)

Una vez creado el modelo que queremos debemos actualizar el proyecto con los siguientes comandos

    - Actualizar proyecto  
        
            python manage.py makemigrations
        
        
 Una vez realizados los cambios los aplicamos con el siguiente comando
  
        
            python manage.py migrate

Podemos obtener mas información a traves de la documentación oficial.
