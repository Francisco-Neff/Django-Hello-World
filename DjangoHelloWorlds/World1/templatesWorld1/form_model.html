Creación de Formularios

Podemos crear formularios a través de "forms.py" ayudándonos de esto podemos crear de una manera muy sencilla nuestros formularios que contengas los datos que hemos creado en el modelo expresado aquí y comprobar que sean validos. 

    - Formulario en forms.py  
        
            class Formulario(forms.ModelForm):
            class Meta:
                model = Modelo1
                fields = ['campo1','campo2']

Una vez creado el formulario podemos validar la respuesta que recibimos por el usuario y el crearlo de una manera muy sencilla al crear la vista. 

    - Ingreso de formulario en la vista  
        
            form = Formulario()
            return render(request, 'form.html',{'form':form})
        
    - Validación del formulario en POST

            form = Formulario(request.POST)
