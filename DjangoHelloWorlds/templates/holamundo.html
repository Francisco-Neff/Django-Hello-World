Creación de un nuevo proyecto en Django

Para crear un nuevo proyecto en Django en Windows se debe ejecutar la siguiente linea de comando en un terminal, siendo Nombre_Proyecto, el nombre con el que queremos crear nuestro proyecto.

    django-admin startproject Nombre_Proyecto

Una vez creado el nuevo proyecto se creara el nuevo directorio de carpetas con el proyecto creado.

        
            Nombre_Proyecto
            ├── Nombre_Proyecto
            ├── db.sqlite3
            └── manage.py

Dentro de la subcarpeta de Nombre_proyecto tendremos una serie de ficheros .py que harán funcionar las vistas de nuestro proyecto, si al crear este no existe un view.py se debe crear para que este quede de la siguiente disposición.

        
            Nombre_Proyecto
            ├── Nombre_Proyecto
            |   ├── settings.py
            |   ├── urls.py
            |   └── views.py
            ├── db.sqlite3
            └── manage.py

Si deseamos ejecutar nuestro proyecto para generar nuestro primer Hola Mundo, tan solo deberemos ejecutar el siguiente comando, estando localizados en el directorio raíz del proyecto.

    python manage.py runserver
