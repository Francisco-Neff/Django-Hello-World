Directorios de recursos

Crear un directorio de plantillas o recursos estáticos nos sirve para tener todos nuestros archivos de cualquier tipo localizados, para ello crearemos los nuevos directorios "template" y "static" como se muestra a continuación

        
            Nombre_Proyecto
            ├── Nombre_Proyecto
            ├── db.sqlite3
            ├── templates
            |   └── recursos
            ├── static
            |   └── recursos
            └── manage.py

En "template" guardaremos todos los "html", mientras que en "static" almacenaremos otro tipos de archivos. Ademas de esto hay que modificar el fichero "settings.py" para vincular los nuevos directorios y el fichero "urls.py" para añadir la vinculación de ficheros estáticos

        - Añadir ruta de template en "settings.py"
        
    STATIC_TEMPLATE = BASE_DIR / 'templates'
    TEMPLATES = [{
    'BACKEND': 'django.template.backends.django.DjangoTemplates',
    'DIRS': [STATIC_TEMPLATE],
    'APP_DIRS': True,
    'OPTIONS': {
        'context_processors': [
            'django.template.context_processors.debug',
            'django.template.context_processors.request',
            'django.contrib.auth.context_processors.auth',
            'django.contrib.messages.context_processors.messages',
        ],},},]
        
    
        - Añadir ruta de static en "settings.py"
        
    STATICFILES_DIRS = [
    BASE_DIR / "static"]
    
        - Añadir ruta de static en "urls.py"
        
    + static(settings.STATIC_URL, document_root=settings.STATIC_ROOT)

Como web de referencia podemos usar la misma documentación que realiza el sitio web Django
